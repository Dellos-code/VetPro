# VetPro Project

 VetPro - Κτηνιατρικό Ψηφιακό Οικοσύστημα

Το VetPro είναι ένα ολοκληρωμένο σύστημα διαχείρισης κτηνιατρικής κλινικής, αναπτυγμένο στα πλαίσια του μαθήματος "Τεχνολογία Λογισμικού" του Τμήματος Μηχανικών Η/Υ και Πληροφορικής (Πανεπιστήμιο Πατρών).

 Σκοπός του Έργου

Σκοπός μας είναι η μετάβαση της κτηνιατρικής μονάδας από τη χειρόγραφη (ή αποσπασματική ψηφιακή) καταγραφή σε ένα έξυπνο οικοσύστημα. Το σύστημα διασφαλίζει την απρόσκοπτη ροή πληροφορίας μεταξύ Ιδιοκτήτη, Κτηνιάτρου και Διοίκησης.

 Αλγοριθμική Καινοτομία

Το σύστημα διαφοροποιείται από απλές εφαρμογές CRUD μέσω δύο κεντρικών μηχανών λογικής (Logic Engines):

Heuristic Scheduler (Βελτιστοποίηση Ραντεβού): Αναδιοργανώνει αυτόματα το ημερήσιο πλάνο σε επείγοντα περιστατικά, ελαχιστοποιώντας τον νεκρό χρόνο της κλινικής.

Stochastic Inventory Forecaster (Πρόβλεψη Αποθεμάτων): Υπολογίζει τον κίνδυνο έλλειψης φαρμάκων (Stockout Risk) και προειδοποιεί προνοητικά για αναπλήρωση.
